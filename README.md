# ResumeRankAI

## AI Resume Screening & Candidate Ranking System

### Project Description

ResumeRankAI is an AI-powered Resume Screening and Candidate Ranking System that automatically analyzes resumes, compares them with Job Descriptions, calculates ATS scores, and ranks candidates using Natural Language Processing (NLP) and Machine Learning.

## Technologies

- Python
- Machine Learning
- NLP
- Flask
- SQLite
- Scikit-Learn
- Pandas
